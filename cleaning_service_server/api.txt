//user routes
POST:/register регистарция пользователя
POST:/login авторизация пользователя
GET:/users получение списка пользователей(администратор)
DELETE:/user/:id (id-идентивикатор пользователя) удаление пользователя(администратор)

//service routes
POST:/services добавление новой услуги(администратор)
GET:/services получения списка услуги
PUT:/services/:id (id-идентификатор услуги) редактирование услуги(администратор)
DELETE:/services/:id (id-идентивикатор услуги) удаление услуги(администратор)

//orders routes
POST:/orders бронирование заказа
GET:/orders получение списка всех заказов пользователя
GET:/orders/:id (id-идентивикатор заказа) получение заказа по id
PUT:/orders/:id (id-идентивикатор заказа) установление, что заказ выполнен
DELETE:/orders/:id (id-идентивикатор заказа) удаление заказа (администратор)

//review  routes
POST:/reviews добавление отзыва по услуге
GET:/reviews получение всех отзывов по услугам
GET:/reviews/:serviceId (id-идентивикатор услуги) получение оценок по конкретной услуге
GET:/review/:id (id-идентивикатор отзыва по услуге)  получение услуги по id
PUT:/reviews/:id (id-идентивикатор отзыва по услуге) рудактирование отзыва
DELETE:/reviews/:id (id-идентивикатор отзыва по услуге) удаление отзыва

//orderRating routes
POST:/orders/:orderId/ratings (orderId-идентивикатор заказа) добавление отзыва по заказу
GET:/orders/:orderId/ratings (orderId-идентивикатор заказа) получение отзывов ко конкретной заказу
GET:/userRatings получение отзывов по услугам пользователя
GET:/ratings получение всех отзывов по выполненым заказам
GET:/ratings/:id (id-идентивикатор отзыва по заказу) получение отзыва по id
PUT:/ratings/:id (id-идентивикатор отзыва по заказу) редактирование отзыва
DELETE:/ratings/:id (id-идентивикатор отзыва по заказу) удаление отзыва
